# Save Our Oceans Website

A simple static website about marine conservation and ocean protection. Created as a pair project for CPE411 to practice Git and GitHub collaboration.

## About the Website

This website promotes ocean conservation awareness with information about marine life protection, ocean cleanup efforts, and ways to get involved. It includes a homepage, about page, get involved page, and contact page. The site is built with HTML, CSS, and JavaScript, featuring a responsive design that works on mobile devices.
